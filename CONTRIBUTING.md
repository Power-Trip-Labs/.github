# How to Contribute

Hello there! If you find any issues or want to submit a pull request, please read below.

## What to do
- Make helpful issues / pull requests.
- Report any bad issues / pull requests.
- Follow the rules of the Code of Conduct.

## What NOT to do
- Submit empty / junk issues or pull requests
- Use it as a way to try to talk to the Code Owners. <br>
  If you want to, join our Discord and open a ticket.
- Spam issues / pull requests.
- You may not use loopholes to bypass these rules.
- You must not encourage others to break the rules too.
- Anything not allowed in the Code of Conduct.
